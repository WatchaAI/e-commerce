# Category Notes

Use these notes to adapt the same workflow to different product types without losing conversion logic.

## 1. Watch / jewelry / eyewear / bag

Prioritize:

- silhouette
- material finish
- hardware details
- wearing scenes
- refined close-ups
- packaging or gift value

Avoid inventing:

- precious-metal purity
- movement / internal structure claims
- certification / appraisal statements unless provided

## 2. Beauty / skincare / makeup / fragrance

Prioritize:

- texture or swatch
- ingredient highlights only when provided
- usage sequence
- suitable skin type or scenario only when supported
- before/after logic only when evidence is provided
- packaging and applicator details

Avoid inventing:

- medical or therapeutic claims
- clinical percentages
- efficacy timelines without source

## 3. Apparel / shoes / accessories

Prioritize:

- cut / fit / silhouette
- fabric texture
- styling combinations
- wearing occasions
- details such as collar, hem, sole, stitching, buckle

Avoid inventing:

- exact fabric percentages if not supplied
- performance claims such as waterproof, thermal, antibacterial without proof

## 4. Food / supplement / beverage

Prioritize:

- taste profile
- ingredient or source story when provided
- serving scenes
- package size and format
- preparation or serving method

Avoid inventing:

- health benefits
- nutrient values
- origin certifications
- regulatory or safety claims beyond supplied evidence

## 5. 3C / electronics / appliances

Prioritize:

- interface layout
- physical design
- use scenarios
- feature modules
- package contents
- parameter sections

Avoid inventing:

- chip model
- battery capacity
- power values
- connectivity standards
- waterproof or durability levels

## 6. Home goods / furniture / lifestyle products

Prioritize:

- material texture
- scale and placement scenes
- touch points
- storage or practical details
- room styling compatibility

Avoid inventing:

- exact dimensions or bearing capacity without data
- eco certification without proof

## 7. Visual rhythm hints by category

- Luxury-like products: darker hero screens, stronger spotlight, less copy
- Daily-consumption products: brighter readability, clearer benefits, stronger function screens
- Technical products: more modular screens and parameter logic
- Giftable products: more packaging and scene-result screens
